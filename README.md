#  AI-Enhanced geometry Control System using CoppeliaSim and MATLAB

This project demonstrates an AI-assisted control system integrated with **CoppeliaSim simulation**, combining classical control (PID), AI-based decision making, and machine learning (LSTM) for stability prediction and performance analysis.

---

#  Project Overview

The system simulates a robotic or aerial platform controlled in real-time using:
- Lua scripts inside CoppeliaSim (low-level control)
- Python scripts (AI + communication + LSTM model)
- Dataset-based learning for stability prediction
- Comparison between PID and AI control methods

---

#  Key Features

-  Real-time communication with CoppeliaSim (Remote API)
-  AI-based control decision system
-  Dataset analysis (load, environment, control type, stability time)
-  LSTM model for predicting system stability
-  Lua-based low-level simulation control
-  PID vs AI performance comparison
